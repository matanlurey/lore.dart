LogSink

LogSink class abstract interface

A low level interface for logging raw messages or bytes to a sink.

This type is intended to be wrapped to provide a more user-friendly API.

A LogSink is not intended to take ownership of the underlying resource, and as such there is no close method. Instead, it is recommended to call LogSink.flush before closing the underlying resource or terminating the program, and close the underlying resource separately (if necessary).

There are two built-in implementations:

  • LogSink.fromBinarySink writes binary data to a Sink<List<int>>.
  • LogSink.fromStringSink writes string messages to a StringSink.
